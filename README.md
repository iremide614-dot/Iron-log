# IronLog — Workout Tracker

A free, installable workout tracker app. Track weights, reps, photos, and personal records.

## Deploy to Netlify with Auto-Updates (Free)

### One-time setup (~10 min):

**1. Create a GitHub account** (skip if you have one)
   - Go to https://github.com/signup
   - Sign up with your email

**2. Create a new repository**
   - Go to https://github.com/new
   - Name it `ironlog`
   - Leave it **Public**
   - Check **"Add a README file"**
   - Click **Create repository**

**3. Upload the app files**
   - On your new repo page, click **"Add file"** → **"Upload files"**
   - Drag ALL the files from this folder into the upload area:
     - `index.html`
     - `app.js`
     - `manifest.json`
     - `sw.js`
     - `icon-192.png`
     - `icon-512.png`
   - Click **"Commit changes"**

**4. Connect to Netlify**
   - Go to https://app.netlify.com and sign up with **your GitHub account**
   - Click **"Add new site"** → **"Import an existing project"**
   - Choose **GitHub**
   - Select your **ironlog** repository
   - Leave "Build command" **empty**
   - Set "Publish directory" to `/`
   - Click **"Deploy site"**

**5. You're live!**
   - Netlify gives you a URL like `random-name-123.netlify.app`
   - Click **"Site settings"** → **"Change site name"** to rename it (e.g., `ironlog`)

### Making updates
After the one-time setup, updating is easy:
1. Go to your GitHub repo
2. Click **"Add file"** → **"Upload files"**
3. Upload the changed files
4. Click **"Commit changes"**
5. Netlify auto-deploys in ~30 seconds. Everyone using the app gets the update next time they open it.

## Install on Phone

**iPhone (Safari):**
1. Open your Netlify URL in **Safari**
2. Tap the **Share** button (square with arrow)
3. Tap **"Add to Home Screen"**
4. Tap **"Add"**

**Android (Chrome):**
1. Open your Netlify URL in **Chrome**
2. Tap the **three-dot menu**
3. Tap **"Install app"**

## Share with Friends

Just send them your Netlify URL! Each person gets their own:
- Workout data (stored on their device)
- Custom splits (they can edit via the ⚙ Splits tab)
- Photos
- Personal records

Nothing is shared between users. No accounts needed.

---

## Features

- **Custom splits** — comes with 4 pre-built, fully editable via the ⚙ tab
- **Quick-load** entire split days with pre-filled weights
- **Camera** — snap pics of machines/exercises
- **Set tracking** — weight, reps, completion
- **History** — browse past workouts with photos
- **Personal Records** — auto-tracked PRs per exercise
- **Installable PWA** — works like a native app
- **Offline support** — works without internet
- **100% free** — no database, no server, no subscriptions
